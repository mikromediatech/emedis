# Mikromed

## What is Mikromed?


## add & remove Module
Go to dir /src/mikromed

add new module 
php mikromed.php buatmodul modulname

delete module 
php mikromed.php hapusmodul modulname

list added module
php mikromed.php list


*deleted modul saved at backup DIR
